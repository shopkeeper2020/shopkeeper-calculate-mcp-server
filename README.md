# Shopkeeper Calculate MCP Server
[中文](README.md) | [English](README_en.md)
## 概述

Shopkeeper Calculate MCP Server 是一个用于数学计算的MCP工具，帮助LLM完成复杂数学公式的计算。仅提供数学表达式的解析工具，而非单独的拆分为+-*/等简单运算。
